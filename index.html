Precautions

  • This website try to solve according to BODMAS rule.
  • Close every bracket that you open.
  • This would not do parentheses matching.
  • Use only " (){}[]. and Number(0-9) "
  • BODMAS Stand for
    • B - Bracket
    • O - Of
    • D - Divide
    • M - Multiplication
    • A - Addition
    • S - Subtration
  • BODMAS rule execute from B to S
  • Here ( O : of ) we have to replace of by × so we directly use ×
